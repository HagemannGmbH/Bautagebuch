HAGEMANN BAUTAGEBUCH V0.3 - TESTVERSION

SO STELLST DU DIE APP ONLINE:

1. Gehe auf vercel.com
2. Melde dich an
3. Klicke auf "Add New Project"
4. Lade den Inhalt dieser ZIP hoch oder importiere das Projekt aus GitHub
5. Klicke auf "Deploy"
6. Nach wenigen Minuten bekommst du einen Link

LOKAL STARTEN:

1. ZIP entpacken
2. Terminal im Projektordner oeffnen
3. npm install
4. npm run dev
5. Browser: http://localhost:3000

WICHTIG:
- Diese Version speichert Berichte lokal im Browser
- Sie ist fuer den ersten Handy-Test gedacht
- Mikrofon funktioniert typischerweise in Chrome oder Edge besser
